# Screenshots

This directory contains the screenshot assets used by the public README and documentation set.

## Available captures

- `login.png` - AccessCore sign-in screen
- `admin-dashboard.png` - Main admin dashboard
- `groups.png` - Group inventory view
- `group-management.png` - Group detail and policy management
- `mfa-enabled.png` - MFA enrollment and enabled state

## Notes

- Files in this directory are embedded directly in GitHub-flavored Markdown.
- The directory is excluded from the Docker build context through `.dockerignore`.
